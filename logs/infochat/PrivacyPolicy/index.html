InfoChat Privacy Policy

Last Updated: October 14, 2025

1. Overview

This Privacy Policy ("Policy") explains how InfoChat ("Bot") collects, uses, stores, and protects user information. By using this Bot, you agree to this Policy.

2. Information We Collect

  • User ID: Used for executing commands and identifying users.
  • Server ID: Used to store the Bot’s settings.
  • Message Content: Collected only when using specific features (e.g., AI conversation history).
  • Other Data: Additional information necessary for certain features (e.g., roles, channel IDs).

3. Purpose of Data Use

  • Providing and improving Bot features
  • Logging and debugging errors
  • Offering personalized features
  • Preventing illegal activities and enhancing security

4. Data Storage and Security

Data is stored on secure servers/databases and is protected using encryption and other security measures.

5. Sharing with Third Parties

The Bot does not share user data with third parties, except for in-game currency-related data. However, data may be disclosed if required by law.

6. Data Deletion Requests

  • Users may request deletion of their data at any time.
  • To request deletion, please contact via Discord DM: gguummaa.
  • Requests will be processed as quickly as possible.

7. Policy Updates

This Policy may be updated as needed. Important changes will be announced in advance. Continued use of the Bot signifies acceptance of the updated Policy.

8. User Responsibilities

  • Users must read and comply with this Policy.
  • Users are responsible for any issues arising from misuse of the Bot.
